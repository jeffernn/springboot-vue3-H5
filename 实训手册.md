# 校园智能打卡与活动积分平台

本平台为学校、学院或班级提供智能化的活动打卡与积分管理服务。通过智能打卡、自动积分、数据统计、排行榜等功能，提升学生参与活动的积极性，实现活动数字化、透明化管理。

## 主要功能

#### 1. **活动管理**

- 活动创建：管理员可发布各类校园活动（如讲座、志愿服务、体育锻炼等），设置活动时间、地点、积分规则等。
- 活动报名：学生可在线报名参加感兴趣的活动。
- 活动签到/打卡：支持二维码打卡、地理位置打卡、蓝牙/NFC打卡等多种打卡方式。
- 活动记录：自动记录每位学生的参与情况、签到时间。

#### 2. **智能打卡**

- 定时打卡：如晨跑、晚自习等按时段自动开放打卡入口。
- 实时验证：防止代打卡，支持人脸识别、定位、蓝牙等防作弊手段。
- 打卡提醒：通过微信/APP/短信等方式提醒学生打卡。

#### 3. **积分管理**

- 积分规则配置：管理员可自定义不同活动的积分获取规则。
- 积分获取：学生参与活动、按时打卡后自动获得相应积分。
- 积分扣除：如缺席、违规可配置积分扣除机制。
- 积分兑换：支持积分兑换奖品、荣誉证书等（可选）。

#### 4. **后台管理**

- 管理员后台：活动发布、积分规则设置、数据导出、用户管理等。
- 数据导出：支持 Excel、PDF 等格式导出活动和积分数据。

#### 5. **移动端支持**(只做H5)

- 支持微信小程序、APP、H5等多端使用，方便学生随时随地打卡。
- 二维码快速打卡，后台自动识别验证。

#### 6. **数据统计与可视化**(可选)

- 活动统计：统计活动参与人数、签到率、积分总量等数据。
- 个人记录：学生可查看自己历史打卡记录、积分明细、兑换历史。
- 班级/院系排行：按积分、参与度等生成排行榜，激励参与。

#### 7. **权限与角色管理**(可选)

- 多角色支持：管理员、教师、学生不同权限分级管理。
- 分组管理：支持按班级、年级、社团等分组管理活动和积分。

#### 8. **通知与消息**(可选)

- 活动通知：自动推送活动发布、打卡提醒、积分变化等消息。
- 异常提醒：如未打卡、打卡失败等自动通知相关人员。

```sql
  `title` VARCHAR(100) NOT NULL COMMENT '活动标题',
  `description` TEXT COMMENT '活动描述',
  `start_time` VARCHAR(100) NOT NULL COMMENT '开始时间',
  `end_time` VARCHAR(100) NOT NULL COMMENT '结束时间',
  `point` INT COMMENT '积分数值',
  `poster` VARCHAR(100) COMMENT '海报',
  `rule_description` VARCHAR(255) COMMENT '积分规则描述',
    
   create_time DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP,   -- 创建时间
   create_by VARCHAR(64) DEFAULT NULL,                        -- 创建人
   update_time DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,  -- 修改时间
   update_by VARCHAR(64) DEFAULT NULL                         -- 修改人
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='活动及积分规则表';
```
